# Employee Attendance & Payroll System

Simple Python application for tracking employee attendance and calculating payroll.

## Overview

This repository contains a single-file Python application named `Employee Attendance & Payroll System.py` that provides basic attendance management and payroll calculation features for small teams.

## Features

- Record employee attendance
- Calculate basic payroll based on attendance
- Save/load data using local files

## Requirements

- Python 3.8 or newer

## Usage

1. Open a terminal in the project folder.
2. Run the script:

```powershell
python "Employee Attendance & Payroll System.py"
```

If you prefer a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python "Employee Attendance & Payroll System.py"
```

## Files

- `Employee Attendance & Payroll System.py` — main application script
